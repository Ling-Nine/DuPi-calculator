# 度派计算器

## 功能特点
基于 SymPy 实现的初级计算器，实现了类LaTeX的排版显示，同时支持导出为LaTeX格式。
## 快速开始

按键<kbd>{≡</kbd>为方程，按键<kbd>|≡|</kbd>,<kbd>[≡]</kbd>,<kbd>(≡)</kbd>,<kbd>{≡}</kbd>为矩阵，<kbd>[+]</kbd>为矩阵增加列，回车增加行。

字母、数字、函数等支持键盘直接输入，并且：
- 按住 <kbd>Shift</kbd> 键入 <kbd>^</kbd> 为上标
- 按住 <kbd>Shift</kbd> 键入 <kbd>_</kbd> 为下标
- 按住 <kbd>Shift</kbd> 键入 <kbd>/</kbd> 为分数
- 上下左右键移动光标

鼠标拖动移动视角。
